# CHATGRAM

## Project Description
CHATGRAM is a modern desktop-focused web platform inspired by Tinder and Omegle that allows users to instantly connect with strangers for text or video conversations in a simple and fast interface.

## Features
- 🔄 Real-time Messaging
- 👥 Random User Matching
- 🎥 Video Chat Support
- 💬 Text Chat
- 📱 Responsive Design
- 🔐 Secure Communication

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Real-time**: Socket.io
- **Hosting**: Render.com

## Installation

```bash
# Clone repository
git clone https://github.com/conectus09/CHATGRAM.git

# Navigate to project
cd CHATGRAM

# Install dependencies
npm install

# Run development server
npm start
```

## Usage
1. Open browser: `http://localhost:3000`
2. Sign up or Log in
3. Get matched with random users
4. Start chatting or video call
